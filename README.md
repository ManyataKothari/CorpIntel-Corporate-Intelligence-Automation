# CorpIntel - Corporate Intelligence Automation

## Overview
A Playwright-based automation tool that extracts company intelligence from ZaubaCorp using CIN numbers and exports structured data to CSV.

## Features
- Automated company lookup using CIN
- Extracts company name, directors, incorporation date, email, capital details, and status
- Handles Cloudflare verification using persistent browser sessions
- Exports results to CSV format

## Tech Stack
- Python
- Playwright
- CSV
- Regular Expressions

## Output Fields
- Company Name
- Date of Incorporation
- Directors
- Email
- Authorised Share Capital
- Paid-up Share Capital
- Current Status
